Quick Start

Try these commands in the terminal:

  • simp pi1.pair(0, s.0) - Evaluate an expression
  • let uncurry(f) = eval . prod(f, I) - Define a function
  • edit - Enter multi-line editing mode (end with semicolon)
  • help - Show all available commands
